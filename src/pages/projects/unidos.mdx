# Unidos Farma: Scalable Design System for a Regional Pharma Brand

## Context
Unidos Farma is a Honduras-based pharmaceutical company preparing to launch a
large-scale line of generic medications across Honduras and Central America.
The brand needed a digital presence that could establish trust at a regional
scale, but the starting point was minimal: a logo, and no existing design
direction. With a one-week timeline, I needed to move from a single brand
asset to a complete, development-ready design system and wireframe.

## Goals
- **Trust-First Visual Language:** Translate a pharmaceutical brand into a
  UI system that reads as credible and clinical, without losing warmth,
  critical for a category where user trust drives conversion.
- **Token-Driven Design System:** Build a fully tokenized color system
  (brand, background, text, secondary, tertiary, outline, and error states)
  so the site can scale across future product lines without redesign.
- **Builder-Ready Handoff:** Structure typography, spacing, and border-radius
  scales as named tokens — gutter, component-gap, section-gap — so the
  system maps directly into WordPress/Elementor global settings during
  development.

## Process

### 1. Reverse-Engineering the Brand System
Starting from Unidos Farma's logo alone, I extracted a full color system
built around their brand red, then layered in supporting palettes: a cool
secondary gray-blue for structure, and a teal tertiary for accenting
product categories like their Cardio line. Every color was defined as a
semantic token — Primary, On Primary, Container, On Container — rather
than a flat swatch, so states like hover, error, and containers stay
consistent across every future page.

![Unidos Farma color tokens and typography scale](/images/projects/unidos-farma-3.webp)

### 2. Typography and Component Foundations
I paired the brand with Outfit, a geometric sans that reads clean and
modern without feeling cold — important for a healthcare-adjacent brand.
The type scale runs from a 48px bold headline down to 14px medium labels,
each named and versioned (Headline XL, Body LG, Label MD) for easy
reference during build. On top of that foundation, I designed the core
components the site would need first: category tags, product cards, and
button variants (primary, secondary, and outline) in their default and
hover states.

### 3. Wireframing for Regional Trust
With the token system locked, I moved into wireframing the core pages,
prioritizing the structural signals that build credibility for a pharma
brand at regional scale: clear product categorization, prominent trust
markers, and a straightforward path from homepage to product line detail.

## Outcome
In one week, I delivered a complete, tokenized Figma design system and
wireframe set — moving Unidos Farma from a single logo to a
development-ready visual language. I'm currently building out the
WordPress implementation myself, using the design system's token structure
to keep the build consistent with the Figma source as the site scales
across new product lines.
